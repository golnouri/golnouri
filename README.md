# معرفی من 👋
علاقه من به دنیای فناوری و برنامه نویسی سال ها پیش از ورود به دانشگاه آغاز شد. از همان دوران، تحلیل سیستم ها، حل مسائل پیچیده و خلق نرم افزارهای کاربردی برای من یکی از جذاب ترین چالش های فکری بوده است. این علاقه به مرور به مسیر حرفه ای زندگی من تبدیل شد. امروزه به عنوان تحلیلگر، طراح و توسعه دهنده نرم افزار فعالیت می کنم و مدیریت پلتفرم «سوکِت» را بر عهده دارم. در طول سال های فعالیت حرفه ای، تجربه طراحی و پیاده سازی انواع سامانه های نرم افزاری، وب اپلیکیشن ها و راهکارهای اختصاصی را کسب کرده ام. اگر برای ایده، کسب و کار یا پروژه نرم افزاری خود به مشاوره، طراحی یا توسعه نیاز دارید، خوشحال می شوم در کنار شما باشم.

## مهارت‌ها

* ASP.NET Core
* C#
* SQL Server

## خدمات قابل ارائه

* توسعه سامانه های تحت وب
* طراحی و توسعه صفحات وب
* تجزیه و تحلیل سیستم ها
* توسعه نرم افزارهای کاربردی تحت ویندوز و اندروید
